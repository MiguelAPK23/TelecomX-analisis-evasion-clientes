# Telecom X – Análisis y evasión de clientes

Proyecto de análisis exploratorio de datos para identificar posibles causas de evasión de clientes (Churn) en la empresa Telecom X.

Este proyecto fue desarrollado como parte del challenge de Data Science de Alura LATAM.

## Objetivo

Analizar el comportamiento de los clientes para identificar patrones asociados con la cancelación del servicio y generar información útil para futuros modelos predictivos.

## Tecnologías utilizadas

- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Análisis realizado

Se analizaron variables clave como:

- Tipo de contrato
- Antigüedad del cliente
- Cargos mensuales
- Método de pago
- Servicios contratados

## Principales hallazgos

- Los clientes con **contrato mensual presentan mayor tasa de cancelación**.
- Los clientes con **menor antigüedad en la empresa tienden a abandonar el servicio con mayor frecuencia**.
- Los **cargos mensuales más altos se asocian con mayor probabilidad de churn**.
